          提交收录说明

          本网站已经开放提交收录，您可以将您的频道/群组提交收录到TGNAV，并获得更多订阅者。

          TGQRY - 由TGNAV打造的Telegram索引机器人，帮助用户方便快捷的查询高质量频道群组，做一个纯净的索引机器人。您可以通过机器人 @TGQRYbot 进行提交收录（原机器人 @tgnavindexbot 已暂停维护）。

          提交要求

          1. 仅收录公开频道/群组；
          2. 灰产，供需，羊毛类不予收录；
          3. 涉及未成年色情，恐怖极端不予收录；
          4. 含有反动言论的频道/群组不予收录；
          5. 刷僵尸粉、阅读量的频道/群组不予收录；
          6. 长期停更的频道不予收录。

          提交步骤

          1. 私聊 @TGQRYbot ；
          2. 点击输入框下方按钮“提交”；
          3. 发送频道/群组链接，格式： https://t.me/TGQRY ；
          4. 点击“修改分类”，选择一个分类；
          5. 点击“修改标签”，添加标签（可选，标签可以优化搜索结果，建议设置）；
          6. 修改频道/群组简介（可选，机器人会自动获取简介）；
          7. 确认信息无误后，点击“提交”按钮；
          8. 等待管理员审核。

          推送更新

          审核通过后，您的频道将同步推送到 收录信息公告频道 。

          人数不少于2000、创建时间不少于3个月的高质量频道，会分批次推送到 精选频道 和网站，特定类别（如NSFW等类别）不会推送到网站。

          特别说明

          请勿在短时间内重复提交收录，否则会被机器人列入黑名单。

          若审核未通过，请仔细核查您的频道是否符合要求，并在改正后重新尝试提交，是金子总会发光！
